# DevTools 面板拉伸与搜索功能

> **提问**：给当前这个组件增加两项能力，1支持可以拉宽它的宽度，并支持宽度记忆，也支持高度，因此就是可以扩大它。2是支持查找任意属性或者任意值，查找到之后可以定位到这个属性或者值的位置，如果有多个属性或者多个值，可支持快捷键查看上一个，下一个。

---

## 回答摘要

### 功能 1：可拖拽调整面板尺寸 + 尺寸记忆

| 要点 | 说明 |
|------|------|
| 拖拽边 | 左、上、左上角、右下角均可拖拽 |
| 最小尺寸 | 320×300px |
| 默认尺寸 | 420×520px |
| 持久化 | localStorage key `zustand-state-monitor-size` |
| 视觉提示 | 右下角 resize 手柄，hover 变主题色 |

### 功能 2：搜索属性/值 + 上下导航

| 要点 | 说明 |
|------|------|
| 搜索范围 | 递归遍历 state 树的所有 key 和 value |
| 高亮样式 | 匹配项黄色高亮，当前活跃项橙色高亮 |
| 自动展开 | 匹配节点的所有祖先路径自动展开 |
| 自动滚动 | 当前活跃匹配项自动 scrollIntoView |
| 快捷键 | Enter 下一个，Shift+Enter 上一个 |
| 计数器 | 显示 `当前/总数` |

### 版本更新

`1.0.8` → `1.1.0`（minor 版本，新增功能）

## 关键结论

- 拖拽调整通过 document 级 mousemove/mouseup 事件实现，避免拖出元素边界时丢失事件
- 搜索功能通过 collectMatches 递归函数收集所有匹配路径，再计算祖先路径自动展开
- 高亮使用 inline style 而非 className，保持项目一致的样式方案
- container 改为 flex 布局以支持固定高度下内容区域自适应滚动
