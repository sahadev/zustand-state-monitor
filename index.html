功能特性

为什么选择 Zustand State Monitor？

实时状态监控

实时追踪所有 Zustand 状态变化，提供详细的变更历史和差异对比，让状态管理一目了然。

自动注册

支持自动注册 Zustand 存储，无需手动配置，开箱即用，极大简化开发流程。

可视化调试

内置美观的开发者工具界面，提供直观的状态查看、历史记录浏览和操作控制功能。

TypeScript 支持

完整的 TypeScript 类型支持，提供类型安全的 API 和智能代码补全，提升开发体验。

选择性监控

支持按需监控特定存储或状态字段，避免性能影响，灵活配置监控范围。

零依赖

除了 React 和 Zustand 外无其他依赖，轻量级设计，不会增加项目负担。

快速开始

三步即可开始使用 Zustand State Monitor

1

安装依赖

使用 npm 或 yarn 安装 Zustand State Monitor 包

npm install zustand-state-monitor
# 或者
yarn add zustand-state-monitor
2

注册存储

将你的 Zustand 存储注册到监控系统中

import { create } from 'zustand';
import { registerExistingStore } from 'zustand-state-monitor';

const useCountStore = create((set) => ({
  count: 0,
  increment: () => set((state) => ({ count: state.count + 1 })),
}));

// 注册存储
registerExistingStore('counter', useCountStore);
3

添加开发工具

在你的应用中添加状态监控开发工具组件

import { StateMonitorDevTools } from 'zustand-state-monitor';

function App() {
  return (
    <div>
      {/* 你的应用内容 */}
      <StateMonitorDevTools />
    </div>
  );
}
